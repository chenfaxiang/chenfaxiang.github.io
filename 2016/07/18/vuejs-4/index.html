在 HTML 使用监听事件 v-on 的好处 ：

1.在 HTML 代码中能够轻松的定位 JavaScript 代码中对应的方法；

2.因为不需要在 JavaScript 里手动绑定事件，ViewModel 代码可以是非常纯粹的逻辑，和 DOM 完全解耦，更易于测试；

3.当一个 ViewModel 被销毁时，所有的事件处理器都会自动被删除，不需要自己去清理；
